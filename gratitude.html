Online Resources & Practice
  • Codeademy - This online learning platform offers free and paid courses on everything from HTML, CSS, Ruby, JavaScript, Git and many more. The courses are well laid out and very accessible, broken down into small chunks of applied learning so that you can see your progress. This is one of the places my coding journey began.
  • freeCodeCamp - This is a community that helps people learn to code in an online learning platform through completing coding challenges, building projects, then contributing to open source projects used by nonprofits. Their recently updated curriculum includes 6 certifications. For more information about curriculum updates, click here.
  • W3Schools - This reference site offers tutorials and references on HTML, CSS, JavaScript, PHP, SQL, Python, W3.CSS, and Bootstrap. It covers most aspects of web programming. My website might not be here today without this helpful resource. Most of the HTML and CSS googling I did to create this website led me back to W3Schools.
  • MDN Web Docs - A learning and reference site for Web technologies and the software that powers the Web such as HTML, CSS, JavaScript and more. I have found this especially helpful when googling for JavaScript topics and explanations.
  • Never underestimate the power of googling!

Podcasts

  • Code Newbie and BaseCS - On Code Newbie, Saron Yitbarek interviews people that share stories from their coding journey in hopes of helping new programmers on theirs. This podcast was one of the first places I heard about other people interested in learning to code. So many inspiring stories and interesting topics are covered here! In it's sister podcast, BaseCS, Vaidehi Joshi and Saron Yitbarek, explore the basics of computer science with topics such as binary, hexing and algorithms. It is based on Vaidehi's blog and covers a lot of helpful information.
  • Coding Bootcamp Podcast - This is Course Report's monthly news roundup of important news across the coding bootcamp industry. It is a great resource for keeping up to date on trends in the larger community of career changing coders.
  • freeCodeCamp - This is the official podcast for the freeCodeCamp community, covering a range of topics from coding tips, advice on learning and the job hunt, explanations of key concepts, sharing developer experiences and more.
  • Syntax - Tasty Web Development Treats - In this podcast, Wes Bos and Scott Tolinski discuss all things JavaScript.

Finding a Nearby Coding Community

  • Code for America Brigade Network - This initiative of Code for America is a national alliance of community organizers, developers and designers that put technology to work in service of their communities. Participants meet monthly to come up with and implement technology projects to benefit their area. It is a great place to get involved in coding projects or even get feedback on one of your own.
  • Girl Develop It - GDI provides affordable programs for women interested in learning web and software development in a judgment-free environment. Through many chapters around the country, they offer short, in-person trainings along with other events. This is a great place to get feedback on a coding project and strengthen your skills.
  • Meetups - Check out a coding Meetup near you. Just search for active groups in your area and find events such as technical trainings, workshops and networking opportunities.

Inspiring Projects and Organizations

  • 180 Website in 180 Days - Jennifer Dewalt tackles the ambitious project of creating 180 websites in 180 days. Seeing her websites and reading her story propelled me toward finally making this website and starting to dive into my own projects.
  • #100 Days of Code - Alex Kallaway created this hashtag-based challenge to help him stay motivated to improve his coding skills and develop a daily habit of coding. He invites others to join the challenge as well - commit to coding 1 hour a day for 100 days, tweet your progress daily and join a community of others doing the same. In this podcast, Alex is interviewed on Code Newbie about the project.
  • Code for America - This organization aims to help make government services simple, effective, and easy to use, working at scale to build healthy, prosperous, and safe communities. They are working with governments to develop services such as improved communication between case managers and their clients on parole, simplifying how Americans access safety net services and helping underserved Californians apply for food assistance. Other projects include managing the Code for America Brigade Network for civicly-minded technologists around the country and a Community Fellowship that pairs local Brigade talent with government to improve services.
